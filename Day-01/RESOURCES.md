# Day 1 — Resources: Linux Fundamentals I

## Primary (assigned)

- **Linux Journey — Grasshopper section** (linuxjourney.com) — free, interactive, the assigned starting point for this day. Covers the filesystem basics in a beginner-friendly, hands-on way.

## Deepen your understanding

- **The Linux Documentation Project — Filesystem Hierarchy Standard**: the closest thing to the "spec" for what goes where.
- **`man hier`** — run this on any Linux box (`man hier`); it's the FHS summarized directly in your terminal. No internet required.
- **Julia Evans' zines** (jvns.ca) — short, illustrated, excellent for building real intuition about how Linux internals work (not specific to today, but a great ongoing companion for this whole phase).
- **explainshell.com** — paste any shell command and get every flag broken down and explained inline. Extremely useful while building command muscle memory.

## Reference / lookup

- `man <command>` and `tldr <command>` — your two on-box references, no internet needed once installed.
- **tldr pages GitHub repo** (github.com/tldr-pages/tldr) — browse community-contributed examples for any command.

## Practice

- **OverTheWire: Bandit** (overthewire.org/wargames/bandit) — a free wargame that teaches Linux fundamentals (including filesystem navigation and `find`) through progressively harder SSH-based challenges. Directly exercises everything in today's lab.
